# Getting Started

### Prerequisites
- Java 17

### API Documentation
http://localhost:8080/swagger-ui/index.html

### Postman
- Import the collection from the file `Booking API.postman_collection.json` in the folder `docs` of the project.
- Create a property before conducting CRUD operations on booking and blocks.
